Atributos usados en la pagina

lang

Define el idioma del documento HTML.

charset

Especifica la codificación de caracteres.

name

Asigna un nombre a un elemento.

content

Proporciona el valor del contenido de un meta tag.

href

Especifica la URL de un enlace.

class

Asigna una o más clases CSS a un elemento.

id

Asigna un identificador único a un elemento.
